# Dentist-SOTA
DentaVision brings the STS-Tooth (2025) paper to life, using an efficient R2 U-Net to spot dental issues in 2D and 3D scans. We’ve built a professional bridge between massive research data and real-world clinics, focusing on speed and ANSI/ADA compliance. It’s about making high-tech diagnostics accessible and ethical for every dentist.
